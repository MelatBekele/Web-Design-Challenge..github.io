Visualizations

The page was created to show the weather changes as you get closer to the equator. To acomplish this analysis, we first pulled data from OpenWeatherMap APIto assemble a dataset on over 500 cities.
After assembling the dataset, we used Matplolip to plot various aspects of the weather vs. latitude. Factors we looked at included: Temparture, Cloudness, wind speed adn humidity. This site provides the source data and visuaizations created as part of the ananlysis, as well as explanation and descriptions of any trends and correlatons witnessed.
